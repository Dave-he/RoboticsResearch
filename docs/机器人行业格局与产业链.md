## 4. 监管与认证

- **ISO 10218** —— 工业机器人安全
- **ISO/TS 15066** —— 协作机器人
- **ISO 13482** —— 个人护理机器人
- **UL 3300** —— 家庭服务机器人(美国,2023)
- **欧盟 AI Act** —— 通用 AI 监管(高风险类别)
- **中国 GB/T 38559** —— 工业机器人安全要求

## 6. 关键不确定性与风险

- **数据瓶颈**:高质量真实操控数据稀缺(sim2real gap)
- **可靠性**:长尾场景下的故障率未达工业可用
- **能源密度**:电池技术限制工作时间(典型 2-4 小时)
- **安全认证**:家庭场景下的 ISO/UL 标准尚未成熟
- **价格 vs 价值**:工业场景 ROI 仍未跑通(除特定重复任务)

## 8. 2026-06-17 增量观察

> 本节为每日研读后追加的新技术信号,待周末合并入正式行

### 8.1 新技术信号

- **VLA 工具调用范式(MuseVLA,2606.17598)**:把多模态传感器(温度 / 声音 / 雷达)重新建模为 LLM 风格的 tool call + grounded sensor image。**对行业的影响**:
  - 主流 VLA(π0 / OpenVLA / Helix)的下一代可能内建 sensor token 词表
  - **多模态传感器 OEM**(FLIR 热像仪 / Velodyne 雷达 / 歌尔声学)成为受益方
  - **传感器融合"中间件"** 出现投资机会(类似 LLM tool-call 编排)
- **可重构轮腿人形(X2-N,2604.21541)**:关节可重构 + 高 DoF + 双形态(人形+轮腿)+ 统一 RL WBC。**对行业的影响**:
  - **国内物流配送**(顺丰 / 美团 / 京东)对楼梯 + 平地混合场景有刚需,X2-N 类形态可能成为"楼梯房家庭服务机器人"主流
  - **国内人形厂商**(智元 / 傅利叶 / 优必选 / 星动纪元)可能跟进可重构关节路线
  - **传统轮腿 OEM**(Ascento / ETH 链)面临换代压力
  - BOM 成本与可重构关节的 MTBF 是关键量产门槛
