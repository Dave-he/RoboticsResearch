# 每日自动化任务使用指南

## 一次性手动运行

```bash
cd /Users/hyx/workspace/RoboticsResearch
python3 scripts/daily_robotics_research.py
```

默认参数:
- `--max-arxiv 80` —— 最多抓 80 篇 arXiv 论文
- `--github-per-query 12` —— 每个 GitHub 查询 12 个结果
- `--min-stars 50` —— 最低 50 stars 过滤

## 启用 macOS launchd 每日定时

(参考 LNN 项目的 `scripts/install_daily_lnn_timer.sh`,可改写为机器人版本)

## 启用 GitHub Actions

(参见 `.github/workflows/daily-robotics-research.yml`)

## 输出位置

每次运行会生成:
- `papers/daily/YYYY-MM-DD_robotics_research.json` —— 原始数据
- `docs/daily/YYYY-MM-DD_robotics_research_digest.md` —— 人类可读摘要
- `analysis/repo_watchlist/YYYY-MM-DD_robotics_repos.md` —— 仓库观察
- 自动更新 `docs/机器人_深度研读报告.md` 全局索引

## 阅读顺序建议

1. 先看 `docs/daily/` 当日摘要
2. 翻到"建议今日深读 (Top 3)"
3. 选 1-3 篇生成研读报告(放在 `docs/reports/`)
4. 在 `docs/机器人_深度研读报告.md` 表格中追加链接
