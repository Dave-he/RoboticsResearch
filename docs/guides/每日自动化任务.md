### 解决方案(挑一个,1-2 分钟搞定)

**方案 A:浏览器刷新 gh token(最简单)**

```bash
gh auth refresh --scopes "repo,workflow,read:org,gist,admin:public_key"
```

会打开浏览器让你重新授权 `workflow` scope。授权后,再:

```bash
cd /Users/hyx/workspace/RoboticsResearch
git add .github/workflows/daily-robotics-research.yml
git commit -m "ci: add daily-robotics-research workflow"
git push -u origin main
```

**方案 B:在 GitHub 网页手动创建**

1. 打开 https://github.com/Dave-he/RoboticsResearch/new/main?filename=.github/workflows/daily-robotics-research.yml
2. 粘贴本仓库 `.github/workflows/daily-robotics-research.yml` 的内容
3. Commit

**方案 C:不配置 Actions,用 macOS launchd 跑本地 cron**

参考 LNN 项目的 `scripts/install_daily_lnn_timer.sh` 改写,完全本地,不需要 GitHub 权限。
