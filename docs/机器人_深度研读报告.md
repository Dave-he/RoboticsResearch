## 📚 全局概览

本项目聚焦于**通用机器人技术**的持续研读,按以下子领域归档:

- 🦵 **人形 / 足式机器人** —— humanoid, locomotion, legged
- 🦾 **操控 / 灵巧手 / 抓取** —— manipulation, grasping, dexterous
- 🧠 **VLA 模型** —— RT-2, OpenVLA, Octo, π0
- 🌐 **具身智能 / 基础模型** —— embodied AI, foundation models
- 🎓 **模仿 / 强化学习** —— imitation learning, RL
- 🗺️ **SLAM / 视觉里程计 / 3D 感知** —— SLAM, NeRF, 3DGS
- 🧭 **导航 / 路径规划** —— navigation
- 🧪 **仿真 / Sim2Real** —— Isaac, MuJoCo
- 🤏 **软体机器人** —— soft robotics
